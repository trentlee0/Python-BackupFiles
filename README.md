## Python-BackupFiles

> 文件复制备份脚本。

### 配置

查看 [latest](https://github.com/trentlee0/Python-BackupFiles/blob/master/src/latest/README.md)

### 运行

注意：先配置 `backup_config.json` 文件，再运行

```shell
cd ./src/latest
python BackupFiles.py
```

也可以指定配置文件运行。

```shell
cd ./src/latest
python BackupFiles.py [confFile]
```
